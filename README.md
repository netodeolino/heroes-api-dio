# Heroes API using WebFlux - Digital Innovation One course


Run project:

```shell script
mvn spring-boot:run 
```

Run tests:

```shell script
mvn clean test
```

Project start endpoint:

```
http://localhost:8080/heroes
```

Project documentation:

```
http://localhost:8080/swagger-ui-reactive-api.html
```

Technologies:

* Java 11
* Maven 3.6.3
* Spring WebFLux
* DynamoDB
* JUnit
* Reactor
